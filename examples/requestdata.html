The Web server passes information to a Go Server Page in the form of a gospReq variable of type GospRequest. gospReq lies in the scope of all <?go:block … ?> and <?go:expr … ?> code. Here are some examples of the data provided via gospReq:

Hello, visitor from host . I see you're interested in the URL . If you experience any problems with this site, please notify .

Much of gospReq's content is provided by the untrusted remote user and should therefore be considered suspect. Use html.EscapeString, the html/template package, or a similar safety mechanism when manipulating data from gospReq.
