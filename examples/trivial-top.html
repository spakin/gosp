Top-level Go code (import, func, etc.) must be enclosed within a <?go:top … ?> tag. Anything declared top-level will be available within <?go:block … ?> and <?go:expr … ?> tags on the same page.

I just wanted to say,

In this example, we placed the <?go:top … ?> tag in the HTML <head> … </head> section, but this is not mandatory. It can appear anywhere on the page.
